# Crop Disease Prediction (ML)

This repository contains a **basic machine learning model** for crop disease prediction.  
The project is currently in an early stage and focuses on setting up the model and application structure.

## Getting Started
Follow the steps below to run the project on your local system.
---

## Installation & Setup
Do all the process in vs code specifically in a new terminal

### 1. Clone the repository

git clone https://github.com/Javagar-S/crop-disease-prediction-ml.git

cd crop-disease-prediction-ml

## Create and activate a virtual environment

## Windows:=====>
python -m venv crop

crop\Scripts\activate

## Linux / macOS:====>  
python3 -m venv crop

source crop/bin/activate

## after creating environment type this 

pip install -r requirements.txt

## Ignore These two Steps for Now
python split_dataset.py

python src/train.py


## Run the Web Application

python web/app.py
